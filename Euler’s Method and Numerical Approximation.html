Euler’s Method and Numerical Approximation

Euler’s method is a way of approximating what a solution curve looks like without having to find a solution to it. This is useful if finding a solution to a particular differential equation is very difficult or just impossible. There are methods of numerical approximation that are more accurate than Euler’s method, but Euler’s method is more intuitive in how it works, so it is a good starting point to learn the idea.

The general idea behind the method is to use segments of the tangent line of a solution at successive points to approximate what a solution curve looks like. Generally, for an ODE \(y'=f(x, y)\) given initial conditions \(x_0, y_0\) and a fixed step size \(h\), in order to obtain an Euler’s method approximation you determine the slope at a point, then follow that slope to the \(x\) value of a step size \(h\) away, and repeat this process as long as desired. The general formulas that summarize this idea: \[x_n=x_0+nh \quad y_{n+1}=y_n+hf(x_n,y_n)\] Example: Use Euler’s method with a step size of \(\pi/2\) to obtain an approximation of \(y'=\cos x\). The initial point is \(y(0)=0\). Calculate the approximation until \(x=\pi\).

For \(h=\pi/2\), the initial point is \((0, 0)\). The next \(x\) point will be \(x+h=0+\pi/2=\pi/2\). The slope of the tangent line at this point can be found using \(y'=\cos x\). \(\cos0=1\). The next \(y\) value can be found by adding the current \(y\) value to \(h\) times the slope of the tangent line at the current point. This results in \(y+hy'=0+(\pi/2)1=\pi/2\). Now the new point is \((\pi/2, \pi/2)\). If we repeat this process for the next point, we obtain the point \((\pi, \pi/2)\). In summary:

\(x\) Approximation Actual Value
\(0\) \(0\) \(0\)
\(\pi/2\) \(\pi/2\) \(1\)
\(\pi\) \(\pi/2\) \(0\)

As you can see, the approximation with a relative step size this large is very bad. The limit as the step size approaches \(0\), will always result in the correct solution curve, but generally other approximation methods will be used instead.

Some ways to quantify the error in the approximation include absolute error and relative error:

absolute error = \(\lvert \text{actual – approximate}\rvert\)

relative error = \(\frac{\lvert \text{actual – approximate}\rvert}{\lvert \text{actual}\rvert}\)

For example, the absolute and relative errors at \(\pi/2\) in the example are \(0.571\) and \(57.1\%\) respectively.
