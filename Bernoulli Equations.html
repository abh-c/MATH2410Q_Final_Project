Bernoulli Equations

A Bernoulli equation has a form similar to a linear equation but has an extra \(y^n\) term multiplied with the \(f(x)\) term, where \(n \ne 0, 1\) because that would allow for simplification down to a linear equation.

\[y' + P(x)y = f(x)y^n\]

Bernoulli equations can be solved using a substitution of \(v(x) = y^{1-n}\). This will always reduce the equation down to a linear ODE, which can be solved using the normal method to solve linear equations. At the end, \(v\) must be substituted with the \(y\) expression to solve the ODE.

Example: Solve the differential equation:

\[y'-2y=2e^{2x}y^2\]

First, the substitution can be determined:

\[v = y^{1-n} = y^{1-2} = y^{-1} \qquad v' = -y^{-2}y'\]

In this case, using the general form of a Bernoulli equation, \(n=2\).

One optional step that makes the substitution simpler is to multiply through by a function \(g(x)\), where \(v' = g(x)y'\). In this case \(g(x) = -y^{-2}\).

\[-y^{-2}y'+2y^{-1}=-2e^{2x}\]

At this point, the substitution can be made:

\[v'+2v=-2e^{2x}\]

This equation is linear, so an integration factor can be used to solve. In this case the integration factor is \(\mu(x) = e^{\int P(x)dx} = e^{\int 2 dx} = e^{2x}\).

\[e^{2x}v'+2e^{2x}v=-2e^{2x}e^{2x}\]

\[\int e^{2x}v'+2e^{2x}v dx=\int-2e^{4x} dx\]

\[e^{2x}v = -\frac{1}{2}e^{4x}+C\]

\[v = -\frac{1}{2}e^{2x}+Ce^{-2x}\]

The final step is to substitute \(v\) for \(y^{1-n}\) to obtain a solution:

\[y^{-1} = -\frac{1}{2}e^{2x}+Ce^{-2x}\]

\[y = \frac{1}{-\frac{1}{2}e^{2x}+Ce^{-2x}}\]
