Extension of Ideas to Higher-Order Equations

In a previous section, homogeneous second-order equations with constant coefficients were discussed. Homogeneous equations with constant coefficients of \(n\)th order more generally take the form: \[c_ny^{(n)} + c_{n-1}y^{(n-1)} + \cdots + c_1y'+c_0y=0\] The solution technique for higher order equations of this form is the same as in the second order case. For example, for a third order equation \(ay'''+by''+cy'+dy=0\), plugging in the solution form \(y=e^{mx}\) results in \(e^{mx}(am^3+bm^2+cm+d)=0\). In this case, the auxiliary equation is: \[am^3+bm^2+cm+d=0\] Since this is a third degree polynomial, there will be three linearly independent solutions. Generally, for an \(n\)th order homogeneous ODE with constant coefficients, there will be \(n\) linearly independent solutions. If all the solutions are unique and real, then the solutions will generally take the form: \[y=c_1e^{m_1x}+c_2e^{m_2x} + \cdots c_ne^{m_nx}\] Fortunately, solutions to higher order equations follow the same pattern as 2nd order equations. The same pattern is also followed if there are repeated roots. Simply make sure that each repeated term as an increasing degree of \(x\) multiplied by it to make sure the solutions are linearly independent.

Example: solve \(y'''+3y''+3y'+y=0\).

The auxiliary equation is \(m^3+3m^2+3m+1=0\). This simplifies to: \((m+1)^3=0 \to m=-1\). This is a triple root, so the general solution is written as \(y=c_1e^{-x}+c_2xe^{-x}+c_3x^2e^{-x}\). If this was a quadruple root, the fourth solution would take the form \(c_4x^3e^{-x}\), and this pattern continues for higher order equations.

At equations of order 4 or higher, it is possible to get double (and greater multiplicity) pairs of complex roots. Fortunately, the same pattern discussed above can be used in these cases. Multiply an \(x\) onto the second \(\cos\beta x\) and \(\sin\beta x\) terms to create the second two linearly independent solutions: for \(m=\alpha\pm\beta i\), \(y=c_1e^{\alpha x}\cos\beta x + c_2e^{\alpha x}\sin\beta x + c_3xe^{\alpha x}\cos\beta x + c_4xe^{\alpha x}\sin\beta x\). If a solution somehow results in a triple or higher complex root, add \(x\) terms of increasing degree to each successive term, like with real roots of higher multiplicity.

Example: solve \(y^{(4)}+2y''+y=0\)

The auxiliary equation is \(m^4+2m^2+1=0\). This factors to \((m^2+1)^2=0\), so there is a double complex root of \(m=0\pm i\). The solution is written: \(y=c_1\cos x+c_2\sin x + c_3x\cos x+c_4x\sin x\).

If you want, you can try using the Wronskian to show that all the solution functions are linearly independent.
