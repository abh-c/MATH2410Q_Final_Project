Separable Equations

Separable equations are first order ODEs where all the \(x\) terms and \(y\) terms can be separated to different sides of the equation, where they can then be integrated to find a solution. Separable equations can be written in the form:

\[\frac{dy}{dx} = g(x)h(y)\]

In this case, \(g(x)\) is a function that only contains \(x\) variables, and \(h(y)\) is a function that only contains \(y\) variables.

In order to solve an equation in this form, one would first have to separate the \(x\) terms and \(y\) terms. For simplicity, the \(\frac{dy}{dx}\) term can be treated as a fraction and separated as well to set up the upcoming integration.

\[\frac{1}{h(y)}\, dy = g(x) \, dx\]

The final step to solve this differential equation is to integrate both sides:

\[\int\frac{1}{h(y)}\, dy = \int g(x) \, dx\]

The result will take a form:

\[k(y) + C_1= l(x) + C_2 \to k(y)= l(x) + C\]

Since the constants \(C_1\) and \(C_2\) are arbitrary, they can be combined into one term. Sometimes, it is possible to solve for \(y\) to obtain an explicit solution. Otherwise, it is fine to leave the equation in the form of an implicit solution.

Example: Solve the differential equation: \[xy'+y=0\]

In this case, the variables can be separated into two terms that multiply each other:

\[\frac{dy}{dx} = \left(\frac{1}{x}\right)(-y)\]

These terms can then be brought to opposite sides of the equation for integration:

\[\int-\frac{1}{y}\,dy = \int \frac{1}{x}\,dx\]

Integrating both sides of the equation results in:

\[-\ln \lvert y \rvert = \ln \lvert x \rvert+C\]

At this point, the problem is solved, and an implicit solution has been found.
