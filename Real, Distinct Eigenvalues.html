Real, Distinct Eigenvalues

Solving for eigenvalues can be used to solve linear homogeneous systems of equations with constant coefficients. For a system of the form: \[\vec{X}'=A\vec{X}\] the general solution is of the form \[\vec{X}(t)=c_1\vec{K}_1e^{\lambda_1 t}+c_2\vec{K}_2e^{\lambda_2 t}\] where the eigenvalues are found using the relationship \[(A-\lambda I)\vec{K}=\vec{0}\] The general solution form written above is what will be obtained if the eigenvalues obtained (2 for a 2 by 2 system) are real and distinct.

Example: solve \[\vec{X}'=\begin{bmatrix} 1&2\\0&2\end{bmatrix}\vec{X}\] Now set up the equation: \[\begin{bmatrix} 1-\lambda&2\\0&2-\lambda\end{bmatrix}\vec{K}=\vec{0}\] Now use the determinant to solve for the eigenvalues \[\begin{vmatrix} 1-\lambda&2\\0&2-\lambda\end{vmatrix}=0=(\lambda-2)(\lambda-1)-0\]

\(\lambda_1=1, \lambda_2=2\)

Now we can use the equation to find an eigenvector corresponding to each eigenvalue: \[\begin{bmatrix} 1-1&2\\0&2-1\end{bmatrix}\begin{bmatrix} x\\y\end{bmatrix}=\vec{0}\to0x+2y=0\] Note: at this point you can check if the \(A-\lambda I\) matrix is correct of the rows are constant multiples of each other. In this case, any vector where \(y=0\) and \(x\) is any real number will be an eigenvector, so just pick an easy one for the solution, like: \(\begin{bmatrix} 1\\0\end{bmatrix}=\vec{K}_1\)

Now do the same for the other eigenvalue: \[\begin{bmatrix} 1-2&2\\0&2-2\end{bmatrix}\begin{bmatrix} x\\y\end{bmatrix}=\vec{0}\to-x+2y=0\] One solution is \(\begin{bmatrix} 2\\1\end{bmatrix}=\vec{K}_2\)

Now, put all the values together in the general solution: \[\vec{X}(t)=c_1\begin{bmatrix} 1\\0\end{bmatrix}e^{ t}+c_2\begin{bmatrix} 2\\1\end{bmatrix}e^{2 t}\]

If desired, linear independence between the two solutions can be determined using the Wronskian. It does not matter which eigenvectors you choose as long as they are linearly independent.
