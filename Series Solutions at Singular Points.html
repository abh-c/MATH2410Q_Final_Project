Series Solutions at Singular Points

For an ODE of the form \(a_2(x)y'' + a_1(x)y' + a_0(x)y = 0\), if the point \(x_0\) is not an ordinary point, then \(x_0\) is a singular point. If you divide through by \(a_2(x_0)\), then you end up with discontinuities. You can only solve regular singular points with the method described here, not irregular singular points. For a general ODE of the form:

\[a_2(x)y'' + a_1(x)y'+a_0(x)y = 0\] divide through by \(a_2(x)\) to get: \[y'' + P(x)y'+Q(x)y = 0\] \(x_0\) is a regular singular point if \(p(x) = xP(x)\) and \(q(x) = x^2Q(x)\) are both analytic at \(x_0\). This means that you will be able to find a series solution because the equation is close enough to being a Cauchy-Euler equation. If these conditions are not true, then you have an irregular singular point and this method will not work.

Frobenius’ Theorem

If \(x_0=0\) is a regular singular point, then there exists at least one solution of the form:

\[y=x^r\sum^\infty_{n=0}c_nx^n=\sum^\infty_{n=0}c_nx^{r+n}\] where \(n \in \mathbb{N}\) (naturals, e.g. \(1, 2, 3, 4, \ldots\)) but \(r \in \mathbb{R}\) (reals, e.g \(0.123, \pi, \sqrt42, \ldots\))

The general procedure to solve these equations is to define the above solution form for \(y\), substitute that value into the ODE, then find a recurrence relation and use it to determine two linearly independent solutions in the form of series.

Example: solve \[3xy''+y'-y=0 \text{ about } x_0=0\] First, define the solution \(y\) as a general power series and find its first and second derivatives:

\[y = \sum^\infty_{n=0}c_nx^{r+n}\]

\[y' = \sum^\infty_{n=0}(r+n)c_nx^{r+n-1}\]

\[y'' = \sum^\infty_{n=0}(r+n)(r+n-1)c_nx^{r+n-2}\]

Note that the initial index has to remain at 0 because it is not guaranteed that any term is 0 because \(r\) can be any real number. This differs from the approach to solving at ordinary points.

Next, substitute into the ODE:

\[\sum^\infty_{n=0}3(r+n)(r+n-1)c_nx^{r+n-1}+\sum^\infty_{n=0}(r+n)c_nx^{r+n-1}-\sum^\infty_{n=0}c_nx^{r+n}=0\] The \(3x\) has been brought into the left sum, which has changed the \(x\) exponent to \(x^{r+n-1}\). Notice that the right sum is being subtracted as well.

Because \(x^r\) only equals \(0\) when \(x=0\)—and that would make the whole solution \(0\), which we are not looking for—an \(x^r\) term can be safely brought out of each sum and divided out:

\[\sum^\infty_{n=0}3(r+n)(r+n-1)c_nx^{n-1}+\sum^\infty_{n=0}(r+n)c_nx^{n-1}-\sum^\infty_{n=0}c_nx^{n}=0\]

Now we can reindex with the goal of getting each \(x\) term to have an exponent of only an index \(k\). For the left and middle sums, we will use the substitution \(k=n-1\), and for the right sum we only have to do \(k=n\):

\[\sum^\infty_{k=-1}3(r+k+1)(r+k)c_{k+1}x^{k}+\sum^\infty_{k=-1}(r+k+1)c_{k+1}x^{k}-\sum^\infty_{k=0}c_nx^{k}=0\]

Now all the \(x\) terms have the same exponent, but to combine the sums the initial indices must be the same as well. To do this, take out the \(-1\)st terms of the first and second sums, then combine the sums. The \(-1\)st term of the left term is \(3r(r-1)c_0x^{-1}\), and the \(-1\)st term of the middle term is \(rc_0x^{-1}\). Combining these terms yields \(r(3r-2)c_0x^{-1}\). After taking these terms out and combining the rest of the sums, the result is:

\[r(3r-2)c_0x^{-1}+\sum_{k=0}^\infty [(k+r+1)(3k+3r+1)c_{k+1}-c_k]x^k=0\] The only case where this equation is true is where the coefficient term in the sum is always equal to zero, since no sum of various powers of \(x\) will ever add with \(r(3r-2)c_0x^{-1}\) to result in \(0\). The coefficient term \((k+r+1)(3k+3r+1)c_{k+1}-c_k\) must equal zero, so the sum must equal zero, so \(r(3r-2)c_0x^{-1}\) must equal \(0\).

For this solution method, we assume \(c_0 \ne0\) because otherwise the whole solution would collapse into nothingness. \(x^{-1}\) can’t equal \(0\) in any case so the so, \(r(3r-2)\) must \(0\). Using this equation \(r(3r-2)=0\), we can find \(r\): \(r = \frac{2}{3},0\).

Also, since the coefficient term in the sum has to equal zero, we can use that to obtain a recurrence relation: \([(k+r+1)(3k+3r+1)c_{k+1}-c_k] = 0\), \(\to c_{k+1} = \frac{c_k}{(k+r+1)(3k+3r+1)}\)

When you plug in both values of \(r\), you get two recurrence relations: \[c_{k+1} = \frac{c_k}{(k+\frac{5}{3})(3k+3)}\] and \[c_{k+1} = \frac{c_k}{(k+1)(3k+1)}\]

Assume \(c_0\) is arbitrary and get \(c_1\) and \(c_2\) in terms of \(c_0\) for both recurrence relations. \(c_0\) is arbitrary and will get absorbed into other constant terms at the end, so you can just assume \(c_0=1\) for convenience.

Some values for the first relation:

\(c_0\) \(1\)
\(c_1\) \(1/16\)
\(c_2\) \((1/16)/33=1/528\)

Some values for the second relation:

\(c_0\) \(1\)
\(c_1\) \(1/8\)
\(c_2\) \((1/8)/21=1/168\)

Now we can substitute the constants we found for both \(r\) values into the original sum we defined for \(y\):

\[r_1: y_1 = \sum^\infty_{n=0}c_nx^{\frac{2}{3}+n} = x^{\frac{2}{3}}+\frac{1}{16}x^{\frac{5}{3}}+\frac{1}{528}x^{\frac{7}{3}}+\cdots\]

\[r_2: y_2 = \sum^\infty_{n=0}c_nx^{n} = x+\frac{1}{8}x^{2}+\frac{1}{168}x^{3}+\cdots\]

In this case, both solutions are defined at zero, but this doesn’t always happen.

You can check the the Wronskian for linear independence.

The general solution is \(y = c_1y_1+c_2y_2\), so it can be written as: \[y=c_1(x^{\frac{2}{3}}+\frac{1}{16}x^{\frac{5}{3}}+\frac{1}{528}x^{\frac{7}{3}}+\cdots)+c_2(x+\frac{1}{8}x^{2}+\frac{1}{168}x^{3}+\cdots)\] where \(y_1\) and \(y_2\) are written as series
