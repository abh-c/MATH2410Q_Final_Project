Variation of Parameters

Variation of parameters is a solution technique that can solve second-order linear ODEs that do not have constant coefficients \[y''+P(x)y'+Q(x)y = g(x)\] as long as it is possible to obtain \(y_c=c_1y_1+c_2y_2\).

Using this method, a solution can be found by using the equations: \[y=u_1y_1+u_2y_2\] \[u_1=-\int\frac{y_2(x)g(x)}{W(x)}dx \qquad u_2=+\int\frac{y_1(x)g(x)}{W(x)}dx\] \(W(x)\) is the Wronskian of \(y_1\) and \(y_2\).

This method is often longer than using other approaches in this section, but it is sometimes necessary to use it, either when the \(y\) terms are multiplied by functions instead of constants or when \(g(x)\) is not made up of types of functions with a finite number of derivatives (e.g. polynomials, sine, cosine, exponentials).

Example: Solve \[y''+y=\csc x\] First, solve for \(y_c\) using the auxiliary equation \(m^2+m=0\). The solution is \(0\pm1i\), so \(y_c=c_1\cos x + c_2 \sin x\).

Next, solve for the Wronskian using the solution functions in \(y_c\): \(\cos x\) and \(\sin x\).

\[W(\cos x, \sin x) = \begin{vmatrix} \cos x & \sin x \\ -\sin x & \cos x \end{vmatrix} = \cos^2x - (-\sin^2 x) = 1\] Note: here, the trigonometric identity \(\sin^2x+\cos^2x=1\) is used to solve the Wronskian.

Next, solve for \(u_1\) and \(u_2\). In this case, \(\cos x\) was used as \(y_1\) and \(\sin x\) was used as \(y_2\). Make sure functions are chosen such that they are consistent with the variation of parameters equations.

\[u_1 = -\int \frac{\sin x \csc x}{1} \, dx = -\int \frac{\sin x}{\sin x} \, dx = -\int 1 \, dx = -x + c_1\]

\[u_2 = +\int \frac{\cos x \csc x}{1} \, dx = \int\frac{\cos x}{\sin x} \, dx= \ln\lvert \sin x\rvert + c_2\] Next, plug in \(u_1\) and \(u_2\) into the solution equation \(y = u_1y_1+u_2y_2\):

\[y = (-x+c_1)\cos x + (\ln\lvert \sin x\rvert + c_2)\sin x\] \[y= c_1\cos x+c_2\sin x - x\cos x + \ln\lvert \sin x\rvert\sin x\] This is the general solution. In this case, \(y_p = - x\cos x + \ln\lvert \sin x\rvert\sin x\).
