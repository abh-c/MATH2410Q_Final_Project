Initial-Value Problems

An Initial-Value Problem (IVP) is when an ODE and initial conditions are given to specify a particular solution curve. For an \(n\)th order ODE, the initial conditions must specify specific values for all derivatives of \(y\) up to and including the \(n-1\)st derivative. It is only when enough initial conditions are given where the constants in the general solution can be solved for to obtain a specific solution curve. For example, for a second order ODE, values of \(y\) and \(y'\) must be given for a solution to be found.

Example: solve \(y''-y=0\qquad y(0)=1,\,\, y'(0)=2\)

This ODE is a homogeneous second-order equation with constant coefficients, so it can be solved using an auxiliary equation. In this case, the equation is \(m^2-1=0\) and the solutions are \(m=\{-1,+1\}\). Therefore, the general solution is: \[y=c_1e^x+c_2e^{-x}\] Now, using the initial conditions, the constant \(c_1\) and \(c_2\) can be found. First, find \(y'\): \[y'=c_1e^x-c_2e^{-x}\] Now the initial conditions can be substituted into \(y\) and \(y'\):

\[y(0)=1 \to \quad1=c_1+c_2\] \[y'(0)=2 \to\quad 2=c_1-c_2\] We have obtained a system of equations and can solve for the constants in a variety of ways. One way is to subtract the bottom equation from the top one to obtain \(-1=2c_2\) and solve for \(c_2=-0.5\) and then substitute that into one of the other equations to obtain \(c_1=1.5\).

Now that the constants are solved, we can obtain the solution to the IVP: \[y=1.5e^x-0.5e^{-x}\] Higher order IVPs can be solved similarly with larger systems of equations
