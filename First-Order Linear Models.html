First-Order Linear Models

Differential equations are used to model a wide variety of phenomena. One subject that can be modeled using a first-order linear model is temperature change over time. One model for this is Newton’s Law of Cooling/Warming: \[\frac{dT}{dt} = k(T-T_m)\] In this case, \(T_m\) is equal to the ambient temperature, and \(k\) is a constant determined experimentally depending on the situation.

Example: A tub of ice cream is warming up while sitting in a room in a way that is modeled by Newton’s Law of Cooling/Warming, where room temperature is \(25\degree C\), and where the constant \(k\) has been experimentally determined to be equal to \(-2\). At time \(t=0\), the ice cream has a temperature of \(0\degree C\). Find the solution \(T(t)\) to the differential equation: \[\frac{dT}{dt} = k(T-T_m)\]

This is a separable equation and can be solved by separating the variables and integrating to find the solution:

\[\frac{1}{T-T_m}dT=k\,dt \to \int\frac{1}{T-T_m}dT=\int k\,dt\]

\[\to \ln\lvert T-T_m\rvert=kt+C\] This is an implicit solution, so technically we could stop here if we only wanted the general solution. However, it isn’t too hard to rearrange this equation to solve for \(T\):

\[e^{\ln\lvert T-T_m\rvert}=e^{kt+C}\] \[\lvert T-T_m\rvert=Ce^{kt}\]

Since the problem stated that he ice cream was “warming up”, we can assume that \(T<T_m\)

\[T_m-T=Ce^{kt}\to T=T_m-Ce^{kt}\] \[T=25-Ce^{-2t}\]

Now to solve for \(C\), we can use the initial condition \((0,0)\):

\[0=25-Ce^0\to C=25\]

The solution modeling this situation is \[T=25-25e^{-2t}\]
