The Inverse Laplace Transform

The inverse Laplace transform for some function \(F(s)\) is defined as: \[\mathscr{L}^{-1}\{F(s)\}=f(t)\] For a function \(F\) for which an inverse transform exists, the inverse transform \(f\) is also guaranteed to be unique.1

Being able to obtain the inverse Laplace transform \(f(t)\) of a function \(F(s)\) is important for solving differential equations and getting an answer in terms of \(t\).

Obtaining the reverse Laplace transform of functions involves recognizing patterns and interpreting what they mean. It is usually helpful to have a table of Laplace transforms handy when doing inverse transforms.

Example: find the inverse Laplace transform of \[F(s)=\frac{2}{2s+5}+\frac{4}{s^2+4}\] The coefficients of these terms can be manipulated to get the proper forms of the transforms of an exponential (left) and sine (right):

\[=\frac{1}{s+5/2}+2\frac{2}{s^2+4}\] Now, we need to follow the pattern of the transform for an exponential function \(\mathscr{L}\{e^{at}\}=1/(s-a)\) and sine \(\mathscr{L}\{\sin(kt)\}=k/(s^2+k^2)\) to perform the inverse transform on both sides: \[f(t)=e^{-\frac{5}{2}t}+2\sin{2t}\]

Note that in some cases, you may need to algebraically manipulate \(F(s)\) in order to preform the inverse transform, like with coefficient manipulation shown in the above example or with partial fraction decomposition.
