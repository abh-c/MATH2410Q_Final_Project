Repeated Eigenvalues

Case 1

The most common case for repeated eigenvalues is when there is a repeated solution for \(\lambda\) and where you can only find one linearly independent eigenvector \(\vec{K}\). By thinking about the matrix \(A\) in \(\vec{X}'=A\vec{X}\) as a transformation. In this case, there is only one span for which vectors stay on the span after the transformation, so you can only find one set of eigenvectors. An example of when this happens is when the transformation looks like a shear, which can be described as one unit vector \(\hat{i}\) or \(\hat{j}\) being held in place while the other moves.

For this case, an additional term must be added to describe a second linearly independent solution: \[\vec{X}_2=\vec{K}te^{\lambda t}+\vec{P}e^{\lambda t}\] where \((A-\lambda I)\vec{P}=\vec{K}\)

The general solution looks like \[\vec{X}(t)=c_1\vec{K}e^{\lambda t}+c_2(\vec{K}te^{\lambda t}+\vec{P}e^{\lambda t})\]

Example: solve \[\vec{X}'=\begin{bmatrix} 1&0\\1&1\end{bmatrix}\vec{X}\] In this case, the eigenvalue is a repeated solution of \(\lambda=1\)

Now we can obtain an eigenvalue: \[\begin{bmatrix} 1-1&0\\1&1-1\end{bmatrix}\vec{K}=\begin{bmatrix} 0&0\\1&0\end{bmatrix}\begin{bmatrix} x\\y\end{bmatrix}=\vec{0}\]

From this, using \(1x+0y=0\), we can determine that \(x=0\) and that \(y\) is free, as long as the vector \(\ne 0\). For simplicity, we can pick \(\vec{K}=\begin{bmatrix} 0\\1 \end{bmatrix}\).

Now, to solve for \(\vec{P}=\begin{bmatrix} x\\y\end{bmatrix}\), solve: \[\begin{bmatrix} 0&0\\1&0\end{bmatrix}\begin{bmatrix} x\\y\end{bmatrix}=\begin{bmatrix} 0\\1 \end{bmatrix}\]

From this, using \(1x+0y=1\), we determine that \(x=1\) and that \(y\) is free. For simplicity, we can pick \(\vec{P}=\begin{bmatrix} 1\\0 \end{bmatrix}\).

Now the general solution will have the form: \[\vec{X}(t)=c_1\begin{bmatrix} 0\\1 \end{bmatrix}e^{t}+c_2(\begin{bmatrix} 0\\1 \end{bmatrix}te^{ t}+\begin{bmatrix} 1\\0 \end{bmatrix}e^{ t})\]

Case 2

The less common case for repeated eigenvalues is the case where the solution for \(\lambda\) is a repeated eigenvalue, but you can determine 2 eigenvectors \(\vec{K}_1\) and \(\vec{K}_2\) that are linearly independent. An example is the system \[\vec{X}'=\begin{bmatrix} 2&0\\0&2\end{bmatrix}\vec{X}\] In this case, the eigenvalues will be a repeated value of \(2\). If we set up the equation to find the eigenvectors, we get: \[\begin{bmatrix} 2-2&0\\0&2-2\end{bmatrix}\vec{K}=\begin{bmatrix} 0&0\\0&0\end{bmatrix}\vec{K}=\vec{0}\] Because the \(A-\lambda I\) matrix is equal to the zero matrix, the eigenvectors can be of any value, so we can pick \(\begin{bmatrix}1\\0 \end{bmatrix}\) and \(\begin{bmatrix}0\\1 \end{bmatrix}\) for simplicity. The general solution would be \[\vec{X}(t)=c_1\begin{bmatrix} 1\\0\end{bmatrix}e^{ 2t}+c_2\begin{bmatrix} 0\\1\end{bmatrix}e^{2 t}=\begin{bmatrix} c_1\\c_2\end{bmatrix}e^{2t}\] You can think about why this case occurs using the matrix \(A\) as a transformation. Because \(A=\begin{bmatrix} 2&0\\0&2\end{bmatrix}\), the coordinate axes donâ€™t rotate at all. This transformation only stretches every vector by an eigenvalue of \(2\). This means that every vector is an eigenvector with eigenvalue 2. This only really happens when under specific transformations, like \(A\).
