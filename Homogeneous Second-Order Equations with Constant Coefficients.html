Homogeneous Second-Order Equations with Constant Coefficients

This section deals with solution techniques for homogeneous linear ODEs. In this case, homogeneous refers to a differential equation that is set equal to 0. All ODEs in this section will have constant coefficients multiplied by the \(y\) terms, so there are no instances of the independent \(x\) variable. Generally they are of the form:

\[ay'' + by' + cy = 0\]

The solutions in these cases will be of the form \(y = e^{mx}\). Even more generally the solution form should be \(y=Ae^{mx}\) but the \(A\) will cancel out because the equation is homogeneous. When plugged into the equation, this results in: \[e^{mx}(am^2+bm+c)=0\] The \(e^{mx}\) term never equals 0, so the \(am^2+bm+c\) term—usually called the auxiliary equation—must equal zero. This fact can be used to solve for \(m\) values in the solutions. There are three possible cases depending on the value of the determinant of the auxiliary equation \(\sqrt{b^2-4ac}\). ^e3bcf0

1. Distinct Real Roots

If \(b^2-4ac \gt 0\), there are 2 unique real roots. The two solutions are \(e^{m_1x}\) and \(e^{m_2x}\), and the general solution is \(y = c_1e^{m_1x} + c_2e^{m_2x}\). These roots will be linearly independent.

2. Repeated Roots

If \(b^2-4ac = 0\) there is 1 real double root. In this case, to make the second solution linearly independent (through reduction of order), an \(x\) term is multiplied with the second solution: \(y = c_1e^{mx} + c_2xe^{mx}\). For higher order equations with triple or greater roots, the power of the \(x\) term is increased for each additional term. It would be \(y = c_1e^{mx} + c_2xe^{mx} + c_3x^2e^{mx}\) for a triple root, for example.

3. Pair of Complex Roots

If \(b^2-4ac \lt 0\) there are 2 complementary complex roots \(m = \alpha \pm \beta i\). Using Euler’s formula, one can derive the general solution form for this case: \(y = c_1e^{\alpha x}\cos(\beta x) + c_2e^{\alpha x}\sin(\beta x)\)

Example 1: Solve \[y''-3y'-10y = 0\] The auxiliary equation is \(m^2-3m-10 = 0\), so the roots are \(2\) and \(-5\), and the general solution is \(y = c_1e^{2x} + c_2e^{-5x}\).

Example 2: Solve \[4y''-4y'+1 = 0\]

The auxiliary equation is \(4m^2-4m+1 = 0\), so there is a single double root of \(1/2\). The general solution is \(y = c_1e^{0.5x}+c_2xe^{0.5x}\).

Example 3: Solve \[y''+y'+y = 0\]

The auxiliary equation is \(m^2 + m+ 1 = 0\), so there are complex roots \(-\frac{1}{2} \pm \frac{\sqrt3}{2}i\). The general solution is \(y = c_1e^{-\frac{1}{2} x}\cos(\frac{\sqrt3}{2} x) + c_2e^{-\frac{1}{2} x}\sin(\frac{\sqrt3}{2} x)\).
