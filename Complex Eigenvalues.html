Complex Eigenvalues

If we solve for an eigenvalue \(\lambda=\alpha+\beta i\) and an eigenvector \(\vec{K}_1=\vec{B}_1+i\vec{B}_2\), then the general solution is: \[\vec{X}(t)=c_1e^{\alpha t}[\vec{B}_1\cos(\beta t)-\vec{B}_2\sin(\beta t)]+c_2e^{\alpha t}[\vec{B}_1\cos(\beta t)+\vec{B}_2\sin(\beta t)]\]

Note: for a complex eigenvalue, its conjugate is also an eigenvalue; for a complex eigenvector, its conjugate is a linearly independent eigenvector

Example: solve \[\vec{X}'=\begin{bmatrix}0&-1\\1&0 \end{bmatrix}\vec{X}\] Solve for the eigenvalue: \[\begin{vmatrix}-\lambda&-1\\1&-\lambda \end{vmatrix}=0\to\lambda^2+1=0\to\lambda=\pm i\] Now solve for an eigenvector: \[\begin{bmatrix}i&-1\\1&i \end{bmatrix}\begin{bmatrix} x\\y\end{bmatrix}=\vec{0}\]

Using the relation \(ix-y=0\to y=ix\), we can choose an eigenvector \(\begin{bmatrix} 1\\i\end{bmatrix}=\begin{bmatrix} 1\\0\end{bmatrix}+i\begin{bmatrix} 0\\1\end{bmatrix}\), where \(\vec{B}_1=\begin{bmatrix} 1\\0\end{bmatrix}\) and \(\vec{B}_2=\begin{bmatrix} 0\\1\end{bmatrix}\). Now using these values and the eigenvalues \(\lambda=0\pm 1i\), we can write the general solution: \[\vec{X}(t)=c_1\left(\begin{bmatrix} 1\\0\end{bmatrix}\cos(t)-\begin{bmatrix} 0\\1\end{bmatrix}\sin( t)\right)+c_2\left(\begin{bmatrix} 1\\0\end{bmatrix}\cos( t)+\begin{bmatrix} 0\\1\end{bmatrix}\sin( t)\right)\]

\[\vec{X}(t)=c_1\left(\begin{bmatrix} \cos t\\-\sin t\end{bmatrix}\right)+c_2\left(\begin{bmatrix} \cos t\\\sin t\end{bmatrix}\right)\]
