Solving a First-Order IVP with the Laplace Transform

Laplace transforms are especially useful for solving initial-value problems. Laplace transforms have the property that \[\mathscr{L}\{f^{(n)}(t)\}=s^nF(s)-s^{n-1}f(0) - \dots - s^1f^{(n-2)}(0)- f^{(n-1)}(0)\] Therefore, we can define the Laplace transforms of the first and second derivatives of a function: \[\mathscr{L}\{f'(t)\}=sF(s)-f(0)\]

By applying a Laplace transform to an ODE, we can use a solution technique that mostly involves algebra instead of calculus. Then we can apply an inverse Laplace transform to obtain the solution, as long as the IVP gives initial conditions at \(t=0\).

Example: solve \(y'-y=0 \qquad y(0)=1\)

First, apply the Laplace transform to both sides of the ODE

\[\mathscr{L}\{y'\}-\mathscr{L}\{y\}=\mathscr{L}\{0\}\] Note that we can take the transforms of \(y'\) and \(y\) and add them because the Laplace transform is linear. We will use \(Y(s)\) to represent the transform of the solution \(y\).

\[\to sY(s)-y(0)-Y(s)=0\] Notice that all we have to do now to solve for \(Y(s)\) is some algebraic manipulation: \[Y(s) =\frac{y(0)}{s-1} = \frac{1}{s-1}\] Now, we can obtain the Inverse Laplace Transform of \(Y(s)\) to obtain the solution: \[\mathscr{L}^{-1}\{Y(s)\}=\mathscr{L}^{-1}\left\{\frac{1}{s-1}\right\}=y(t)=e^t\] At this point, we have solved the IVP. You can check the initial condition to check your answer.

Note: often performing the inverse transform requires using partial fraction decomposition to make \(Y(s)\) the proper form to perform the inverse transform.

Laplace transforms are often useful in solving equations that may require more steps using other techniques. If you prefer algebra to calculus, then consider using the Laplace transform when possible.
