Linear Equations

Linear equations have the general form:

\[a_n(x)y^{(n)} + a_{n-1}(x)y^{(n-1)} + \dots + a_1(x)y' + a_0(x)y = b(x)\]

First order linear ODEs have the form:

\[a_1(x)y' + a_0(x)y = b(x)\]

and, since \(a_1(x) \ne 0\) by definition, one can divide by \(a_1(x)\) to simplify them into the form:

\[y' + P(x)y = f(x)\]

The general strategy to solve first order linear equations is to multiply both sides with an integrating factor \(\mu(x)\).

\[\mu(x) y' + \mu(x)P(x)y = \mu(x)f(x)\]

The integrating factor is chosen such that \(\mu(x)P(x) = \mu'(x)\). For this to be true:

\[\mu(x) = e^{\int P(x) \, dx}\]

Note: You do not need to worry about adding a constant term for this integration because it should be able to be cancelled out after multiplying both sides of the equation by the integrating factor.

This will lead to an expression of the form:

\[\mu(x) y' + \mu(x)'y = \mu(x)f(x)\]

Because the left side of the equation takes the form of the product rule, taking the integral of both sides results in an an equation of the form:

\[\mu(x)y = \int\mu(x)f(x)\,dx\]

At this point an explicit or implicit solution can be found.

Example: Solve the differential equation: \[y' + 10y = e^x\]

This differential equation is a first order linear ODE in the form \(y' + P(x)y = f(x)\), so it is possible to use an integrating factor to solve. First one should obtain the integrating factor:

\[\mu(x) = e^{\int P(x) \,dx} = e^{\int10\,dx} = e^{10x}\]

Then, multiply the integrating factor on both sides of the equation:

\[e^{10x}y'+10e^{10x}y = e^{10x}e^x\]

Then, integrate both sides:

\[\int e^{10x}y'+10e^{10x}y \,dx= \int e^{11x}\, dx\]

\[e^{10x}y = \frac{1}{11}e^{11x} + C\]

The left side can be solved by reversing the product rule, and the right side can be integrated normally. At this point, a solution has been found:

\[y = \frac{1}{11}e^x+Ce^{-10x}\]
