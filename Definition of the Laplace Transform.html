Definition of the Laplace Transform

The purpose of a Laplace transform is to convert an ODE from a problem that requires calculus to solve to a problem that only requires algebra to solve.

A Laplace transform of a function \(f(t)\) that is defined at \(t \ge 0\) is defined as:

\[\mathscr{L}\{f(t)\}=\int_0^\infty e^{-st}f(t)dt=F(s)\]

Note that a Laplace transform \(F(s)\) of \(f(t)\) is only guaranteed to exist for some interval of \(s\) if \(f(t)\) is of exponential order, meaning there exists some general exponential function \(Me^{ct}\) that at some point becomes greater than \(\lvert f(t) \rvert\) for some \(t\gt0\). This is important because the Laplace transform includes an improper integral with a bound at infinity. If \(f(t)\) is of exponential order then the \(e^{-st}\) term will cause the integral to converge and exist on some interval. If \(f(t)\) is greater than exponential order, then it can be thought of as “outpacing” the exponential \(e^{-st}\) in growth and not guaranteeing that the integral at infinity will converge.

The Laplace transform is a linear process, so \(\mathscr{L}\{af(t)+bg(t)\}=a\mathscr{L}\{f(t)\}+b\mathscr{L}\{g(t)\}\). Note that the transform of a product of two functions is not equal to the product of their transforms; the transform must be done out by the definition. Another property is that the Laplace transform of \(0\) is \(0\). The value of \(s\) can be complex, but complex functions are often just written as \(\mathscr{L}\{f(t)+ig(t)\}=\mathscr{L}\{f(t)\}+i\mathscr{L}\{g(t)\}\) using linearity.

Example: find the Laplace transform of an exponential function \(e^{at}\)

\[\mathscr{L}\{e^{at}\}=\int_0^\infty e^{-st}e^{at}dt\] \[=\int_0^\infty e^{-(s-a)t}dt=-\frac{1}{s-a}e^{-(s-a)t}|_0^\infty\]

\[\mathscr{L}\{e^{at}\}=\frac{1}{s-a}\] assuming \(s\gt a\)

Some notable transforms: - For functions of the form \(t_n\), \(\mathscr{L}\{t^n\}=\frac{n!}{s^{n+1}}\) where \(n\) is a natural number (e.g \(1. 2. 3, \ldots\)) - \(\mathscr{L}\{\cos(kt)\}=\frac{s}{s^2+k^2}\) - \(\mathscr{L}\{\sin(kt)\}=\frac{k}{s^2+k^2}\)
