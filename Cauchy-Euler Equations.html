Cauchy-Euler Equations

Cauchy-Euler Equations are linear ODEs of the form: \[ax^2y'' + bxy'+cy = g(x)\] For Cauchy-Euler equations, \(y_c\) will be of the form \(x^m\) instead of \(e^{mx}\) like in equations with constant coefficients. Substituting this into the homogeneous equation results in:

\[x^m(am^2+(b-a)m+c) = 0\] To make things simpler, we can assume \(x \gt 0\) to get the auxiliary equation \(am^2+(b-a)m+c = 0\).

Solutions could be defined at \(x=0\), but are guaranteed to be defined at \(x \gt 0\). If \(x\) is not guaranteed to be positive by the problem, then substitute all instances of \(x\) with \(\lvert x\rvert\).

Depending on the number and type of solutions (indicated by the discriminant), there are 3 solution cases:

1. Distinct Real Roots

If \(\sqrt{b^2-4ac} > 0\), there are 2 distinct real roots \(m_1\) and \(m_2\).

\[y_c = c_1x^{m_1} + c_2x^{m_2}\]

2. Repeated Root

If \(\sqrt{b^2-4ac} = 0\), there is 1 repeated root \(m\). \[y_c = c_1x^{m} + c_2x^{m}\ln{x}\] In order to make the second solution linearly independent from the first, it is necessary to multiply by \(\ln x\). This is because because multiplying by just \(x\) will work to create linear independence when solutions are of the form \(e^{mx}\) but not when solutions are of the form \(x_m\).

3. Pair of Complex Roots

If \(\sqrt{b^2-4ac} \lt 0\), there is a pair of complex roots \(m = \alpha \pm \beta i\). Using the fact that \(x^{\alpha + \beta i} = x^\alpha e^{(B\ln{x})i}\) and Euler’s formula \(e^{i\theta} = \cos\theta + i\sin\theta\), one can obtain a solution form:

\[y_c = c_1x^\alpha\cos(\beta\ln(x)) + c_2x^\alpha\sin(\beta\ln(x))\]

After obtaining \(y_c\), to solve for \(y=y_c+y_p\), put the equation into the form \(y''+P(x)y'+Q(x)y = g(x)\) and solve using variation of parameters.

Example: Solve \[x^2y''-5xy'+8y = x^5 \quad \text{assume} \; x \gt 0\] The auxiliary equation for this Cauchy-Euler equation is \(m^2+(-5-1)m+8 = 0\). The solutions are \(m = 2, 4\), so \(y_c=c_1x^2+c_2x^4\).

Next, find the Wronskian for \(y_1=x^2\) and \(y_2=x^4\):

\[W(x^2,x^4) = \begin{vmatrix} x^2 & x^4 \\ 2x & 4x^3 \end{vmatrix} = 4x^5-2x^5=2x^5\] Next, to find \(g(x)\), the ODE must be put into the form \(y''+P(x)y'+Q(x)y=g(x)\). In this case, this can be done by dividing through by \(x^2\):

\[y''-\frac{5}{x}y'+\frac{8}{x^2}y=x^3\] In this form, it is clear that \(g(x)=x^3\). Now, using the variation of parameters equations, \(u_1\) and \(u_2\) can be found:

\[u_1 = -\int \frac{x^4 x^3}{2x^5} \, dx = -\int \frac{1}{2}x^2 \, dx = -\frac{1}{6}x^3 + c_1\] \[u_2 = +\int \frac{x^2 x^3}{2x^5} \, dx = \int \frac{1}{2} \, dx = \frac{1}{2}x + c_2\]

Finally, these expressions can be plugged into \(y=u_1y_1+u_2y_2\) to obtain the solution:

\[y = \left(-\frac{1}{6}x^3 + c_1\right)x^2+\left(\frac{1}{2}x + c_2\right)x^4\] \[y = c_1x^2+c_2x^4+\frac{1}{3}x^5\] In this case, \(y_p = \frac{1}{3}x^5\).
