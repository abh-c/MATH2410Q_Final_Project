The Method of Undetermined Coefficients

The method of undetermined coefficients will be used to solve second order non-homogeneous linear equations with constant coefficients: \(ay'' + by' + cy = g(x)\). \(g(x)\) must also be some sum or product of exponential, polynomial, sine, and cosine functions—functions with a finite list of derivatives.

The general solution will take the form \(y = y_c+y_p\). \(y_c\) is the “complementary” solution, which is obtained by making the equation homogeneous (setting \(g(x) = 0\)), and solving the homogeneous equation with constant coefficients. \(y_p\) is the “particular solution” whose form depends on which functions make up \(g(x)\).

In order to solve for \(y_p\), one must assume its form to be the “general” form of the types of functions within \(g(x)\). For example: - if \(g(x) = 13x\), then \(y_p\) takes the form \(Ax+B\), a generic linear polynomial - if \(g(x) = 123x^2+1\), then \(y_p\) takes the form \(Ax^2+Bx+C\), a generic quadratic - if \(g(x) = \cos10x\) or \(\sin10x\), then \(y_p\) takes the form \(A\cos10x+B\cos10x\), a generic sinusoid - if \(g(x) = e^{-50x}\), then \(y_p\) takes the form \(Ae^{-50x}\), a generic exponential of a specific power

Combinations of these functions work similarly. For example, if \(g(x) = xe^x\), then \(y_p\) takes the form \((Ax+B)e^x\). In this case the exponential constant can be distributed and assimilated into \(A\) and \(B\).

Once the form of \(y_p\) is obtained, it should be “matched” with the terms of \(y_c\). If some terms match, then \(x\) is multiplied with the corresponding \(y_p\) term. The purpose of this process, similar to finding solutions when the auxiliary equation has double roots, is to create a solution for \(y_p\) that is linearly independent from \(y_c\).

For example, if \(y_c = c_1x\) and \(y_p\) is of the form \((Ax+B) + Ce^x\), then the \(Ax\) term “matches” the form of \(c_1x\). You could then multiply the first term by \(x\) to get \(x(Ax+B) + Ce^x\). Now the new \(Ax^2\) term is linearly independent of the \(c_1x\) term. However, when you distribute, you get a \(Bx\) term, which is still the same form as \(c_1x\). You have to multiply by \(x\) again to get \(x^2(Ax+B) + Ce^x\). Now none of the terms match the form of any term in \(y_c\), so the solution form has been fixed.

After finding the form of \(y_p\), final step is to determine expressions for \(y_p'\) and \(y_p''\) and plug them into the ODE. Solve for the constants to obtain the specific solution, then write the final solution as \(y = y_c+ y_p\).

Example: Solve \[y''-4y'+4y=10e^{2x}\] The first step is to solve for \(y_c\). The homogeneous equation is \(y''-4y'+4y=0\), so the auxiliary equation is \(m^2-4m+4=0\). The solution in this case is a double root \(m=2\), so: \[y_c=c_1e^{2x} +c_2xe^{2x}\]

The next step is to find the form of \(y_p\) depending on \(g(x) = 10e^{2x}\). In this case, it should be some exponential with an exponent of \(2x\), so we can say that: \[y_p = Ae^{2x}\]

After this, we can “match” the terms in \(y_p\) and \(y_c\) to make sure \(y_p\) is a linearly independent solution. \(Ae^{2x}\) is of the same form as \(c_1e^{2x}\), so we can multiply \(y_p\) by \(x\) to get \(Axe^{2x}\). However, \(Axe^{2x}\) is of the same form as \(c_2e^{2x}\), so we can increase the power of the \(x\) to get \(Ax^2e^{2x}\). Now \(y_p\) is in a form that is linearly independent of any term in \(y_c\). After this, we can find the first and second derivatives of \(y_p\):

\[y_p = Ax^2e^{2x}\]

\[y_p' = 2Axe^{2x}+2Ax^2e^{2x}\]

\[y_p'' = 2Ae^{2x}+8Axe^{2x}+4Ax^2e^{2x}\]

Now we can plug these values into the ODE as a solution to solve for \(A\):

\[2Ae^{2x}+8Axe^{2x}+4Ax^2e^{2x}-4(2Axe^{2x}+2Ax^2e^{2x})+4Ax^2e^{2x} = 10e^{2x}\] \[2Ae^{2x} = 10e^{2x}\]

\[2A = 10 \to A = 5 \to y_p = 5x^2e^{2x}\] Notice how most of the terms cancelled out and made finding \(A\) relatively simple. If this does not occur, check if you wrote \(y_p\) in the correct form, or if you made an algebraic error.

Now we can write the solution \[y = y_c+y_p = c_1e^{2x} +c_2xe^{2x}+5x^2e^{2x}\]
