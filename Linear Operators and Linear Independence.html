Linear Operators and Linear Independence

Linear operators are functions that perform some kind of transformation on input functions and have certain properties. For example, \(L[f(x)+g(x)] = L[f(x)] + L[g(x)]\), and \(L[Cf(x)] = CL[f(x)]\). Derivatives and integrals have these linear properties.

One example of a linear operator is \(L\), which can be used to represent a generic linear equation with less text: \[a_n(x)y^{(n)} + a_{n-1}(x)y^{(n-1)} + \cdots + a_1(x)y'+a_0(x) y = L[y] = 0\] Functions are said to be linearly independent if, for some interval, the an expression cannot be written such that a combination of the functions equals 0:

\[c_1f_1(x)+c_2f_2(x)+c_3f_3(x) + \cdots + c_nf_n(x) = 0\]

where one or more of the constants \(\ne\) 0.

For example, the functions \(x\) and \(2x\) would be linearly dependent because one is a multiple of the other (\(-2x+2x=0\)). Functions like \(x\) and \(x^2\) are linearly dependent because there is no constant multiple combination that will add up to 0.

Linear independence relates to the fundamental set of solutions to a differential equation. If a set of linearly independent functions are solutions to a differential equation, then a constant multiple combination of those functions make up the fundamental set of solutions for an \(n\)th order differential equation:

\[y = c_1y_1 + \cdots + c_ny_n\]

All of these solutions are unique and can be combined to create new solutions.

The Wronskian

One way to check for linear independence is the Wronskian, which is defined as follows: \[W(f_1,f_2,\ldots, f_n) = \begin{vmatrix} f_1 & f_2 & \cdots & f_n \\ f_1' & f_2' & \cdots & f_n' \\ \vdots & \vdots & \ddots & \vdots \\ f_1^{(n-1)} & f_2^{(n-1)} & \cdots & f_n^{(n-1)} \end{vmatrix}\]These functions are linearly independent on some interval if and only if the Wronskian \(\ne\) 0 on that interval.
