Unit 1 – An Introduction to Differential Equations

Basic Terminology

Differential equations

Equations that include derivatives of unknown functions – often used for modeling applications

Example: \(5x^2y'+2xy = 0\)

Ordinary Differential Equations (ODEs)

Differential equations where there is only one independent variable

Example: \(\frac{d^2x}{dt^2} + 4\frac{dy}{dt} = 2\)

Partial Differential Equations (PDEs)

Differential equations containing partial derivatives of a function and there are at least two independent variables

Example: \(\frac{\partial T}{\partial t} = 2\frac{\partial^2T}{\partial x^2}\)

The Order of a Differential Equation

The order of a differential equation is equal to the order of the highest derivative

Example: The order of \(\frac{d^2x}{dt^2} + 4\frac{dy}{dt} = 2\) is 2 because the highest derivative is a second derivative

Normal Form

An \(n\)th order ODE can be written as:

\[F(x,y,\frac{dy}{dx},\frac{d^2y}{dx^2},\dots,\frac{d^ny}{dx^n})=0\]

If the \(n\)th derivative is continuous and real, then it can be written in normal form:

\[\frac{d^ny}{dx^n}=F(x,y,\frac{dy}{dx},\frac{d^2y}{dx^2},\dots,\frac{d^{n-1}y}{dx^{n-1}})\]


Solutions to Differential Equations

A solution \(\phi\) to a differential equation has derivatives that are continuous on some interval, for which \(F(x, \phi, \phi',\dots,\phi^{(n)})=0\) within the interval.

Some solutions belong to a solution family where one or more constants can vary and still satisfy the differential equation

If a solution doesn’t belong to a family, then it is a singular solution

If \(y=0\) is a solution, it is often called a trivial solution

Solutions are defined on an interval of definition, which contains the values of the independent variable for which the solution is defined. For example, if an ODE had a solution \(y = 1/\sqrt{x}\), then the interval of definition would be \(x \in (0,\infty)\) because that is where \(y\) is defined.

Explicit Solutions

Explicit solutions contain only the dependent variable

Example: \(y = x^2+x\)

Implicit Solutions

Implicit solutions cannot be written solely in terms of the dependent variable

Example: \(y^3+y^2=x^4+3x+C\)

Linearity

An ODE is linear if all its coefficients of derivatives are only functions of the independent variable

Linear example: \(3xy'+xy = \ln x\)

Non-linear example: \(y\tan y + e^y = x^2\)


Initial Value Problems (IVPs)

An IVP is a differential equation with order \(n\) with specified values of the unknown function and its derivatives up to \(n-1\) at a certain point. The purpose of an IVP is to obtain a specific solution to the differential equation. Solutions do not need to be unique.

Existence and Uniqueness Theorem

For an IVP, if \(f\) (the function in normal form) and \(\frac{\partial f}{\partial y}\) (its partial derivative with respect to \(y\)) are both continuous in some rectangle around the initial value \((x_0, y_0)\), then there must exist a unique solution to the IVP on some interval within that rectangle.

If only \(f\) is continuous in the rectangle, then a solution is guaranteed to exist within the interval, but is not guaranteed to be unique.

If neither \(f\) nor \(\frac{\partial f}{\partial y}\) are continuous in the rectangle, then a unique solution could exist but is not guaranteed to exist by the theorem.

Example: \(y' = y^{1/2}, y(1)=1\)

At the point \((1,1)\) both \(y' = y^{1/2}\) and \(\frac{\partial y'}{\partial y} = \frac{1}{2y^{1/2}}\) are continuous, so there exists a unique solution for this IVP on some interval around \((1,1)\).

If instead the initial point was \((0,0)\), then \(y'\) would be continuous but \(\frac{\partial y'}{\partial y}\) would not because there would be a division by \(0\). This means that a solution exists at \((0,0)\) but it is not guaranteed to be unique.
