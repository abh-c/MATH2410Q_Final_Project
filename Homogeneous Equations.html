Homogeneous Equations

A homogeneous function is defined as a function in which all terms have equal total degree. The general definition is: \[f(tx, ty) = t^\alpha f(x, y)\] An example is \(f(x) = x^2y + y^2x\). The total degree of each term \(3\) is equal, so the function is homogeneous. An example of a non-homogeneous function is \(f(x) = x^3+y^2\). In this case, the degree is one term \(3\) does not equal the degree of another term \(2\).

Note: the definition of “homogeneous” used in this section does not refer to a case where some ODE equals 0. That is a different definition using the same word.

A homogenous equation can be written in the form \[M(x, y)\, dx + N(x,y)\, dy = 0 \quad \text{or} \quad \frac{dy}{dx}=\frac{M(x,y)}{N(x,y)}\]

where \(M\) and \(N\) are both homogeneous functions with equal total degree \(\alpha\).

Homogeneous equations can be solved using the substitution \(u = \frac{y}{x}\), where \(u\) is a function of \(x\). This will always convert the equation in to a separable equation. After solving the separable equation, \(u\) can be substituted with \(\frac{y}{x}\) to solve for \(y\).

Example: Solve the differential equation:

\[\frac{dy}{dx} = \frac{2y-x}{y+2x}\]

The total degrees of \(M\) and \(N\) in this case are both \(2\), so this is a homogeneous equation and can be solved through substitution.

\[y = ux \qquad \frac{dy}{dx} = \frac{du}{dx}x+u\]

\[\to \frac{du}{dx}x+u = \frac{2ux-x}{ux+2x} = \frac{2u-1}{u+2}\]

This equation can now be rearranged to create a separable equation:

\[\frac{du}{dx}x = \frac{2u-1}{u+2} - u = \frac{2u-1}{u+2} - \frac{u^2+2u}{u+2}=\frac{-u^2-1}{u+2}\]

\[\frac{du}{dx}x=-\frac{u^2+1}{u+2}\]

\[-\frac{u+2}{u^2+1}\,du = \frac{1}{x}\,dx \to \int\frac{u+2}{u^2+1}\,du = \int-\frac{1}{x}\,dx\]

\[\frac{1}{2}\ln(u^2+1)+2\arctan u = -\ln x + C\]

At this point, algebraic manipulation can be used to further simplify this solution, but it is also fine to substitute in \(u = \frac{y}{x}\) and obtain an implicit solution.

\[\frac{1}{2}\ln(\frac{y^2}{x^2}+1)+2\arctan \frac{y}{x} = -\ln x + C\]
